Canvas clock

This is jquery ui widget. If you want to add this widget on your page you need.

  1. Download this script
  2. Add in you html document this scripts
    • <script src="http://ajax.googleapis.com/ajax/libs/jquery/1.8.2/jquery.min.js"></script>
    • <script src="http://ajax.googleapis.com/ajax/libs/jqueryui/1.8.23/jquery-ui.min.js"></script>
    • <script src="Path to you js file/canvas-clock.js"></script>
  3. Add html elemnt into your page. This will containter for widget. On onload script write code like this
    • $("element").canvasClock({radius : 100});
  4. Enjoy!
